# Data Analyst Portfolio Website

A modern, responsive portfolio website designed specifically for data analysts to showcase their skills, projects, and expertise.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices (desktop, tablet, mobile)
- **Modern UI**: Clean and professional design with smooth animations and transitions
- **Data Analyst Focused**: Tailored sections highlighting data analysis skills and projects
- **Interactive Elements**: Animated skill bars, scroll animations, and interactive navigation
- **Contact Form**: Built-in contact form for potential employers/clients to reach out

## Sections

1. **Hero Section**: Eye-catching introduction with call-to-action buttons and social links
2. **About**: Personal introduction with key statistics and achievements
3. **Skills**: Comprehensive display of technical skills including:
   - Programming languages (Python, SQL, R)
   - Data visualization tools (Tableau, Power BI, Matplotlib)
   - Tools & libraries (Pandas, NumPy, Excel, Git)
   - Analytics capabilities (Statistical Analysis, A/B Testing, Predictive Modeling)
4. **Projects**: Showcase of 6 featured data analysis projects with metrics and technologies used
5. **Contact**: Contact form and information for getting in touch

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- JavaScript (Vanilla)
- Font Awesome Icons

## How to Use

1. Open `index.html` in a web browser to view the portfolio
2. Customize the content by editing `index.html`:
   - Replace placeholder text with your own information
   - Add your actual projects and achievements
   - Update contact information and social links
3. Modify styling in `styles.css` to match your personal brand
4. Enhance functionality in `script.js` as needed

## Customization Guide

### Updating Personal Information

Edit the following sections in `index.html`:
- Hero section: Update name and tagline
- About section: Add your bio and statistics
- Skills section: Adjust skill levels and add/remove skills
- Projects section: Replace with your actual projects
- Contact section: Update email, location, and phone number
- Social links: Add your LinkedIn, GitHub, Kaggle profiles

### Changing Colors

Modify the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #3b82f6;
    /* ... other color variables */
}
```

### Adding Projects

To add more projects, copy the `.project-card` structure in the HTML and customize:
- Project icon
- Title and description
- Technologies used (tags)
- Key metrics
- Project link
